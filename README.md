# Employee Payroll System (Java)

The Employee Payroll System is a terminal-based Java application designed to demonstrate the implementation of Object-Oriented Programming (OOP) concepts in a real-world scenario. This project simulates a simple payroll management system that handles different types of employees and calculates their salaries using abstraction and inheritance.

## Features

- Abstract Employee class as the base for all employee types  
- FullTimeEmployee class with fixed monthly salary calculation  
- PartTimeEmployee class with hourly wage-based salary calculation  
- Terminal-based interface for easy interaction  
- Demonstrates key OOP principles in Java  

## OOP Concepts Used

**Encapsulation** – Protecting employee data using private variables and getters/setters  

**Inheritance** – Creating specialized employee classes from a base class  

**Abstraction** – Using abstract classes and methods  

**Polymorphism** – Different implementations of salary calculation methods  

## Technologies Used

- Java  
- Object-Oriented Programming (OOP)
